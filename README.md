### Steps

1. Prepare min. ubuntu server >= 14.04
2. Install go lang

    sudo apt-get -y upgrade
    sudo curl -O https://storage.googleapis.com/golang/go1.6.linux-amd64.tar.gz
    sudo tar -xvf go1.6.linux-amd64.tar.gz
    sudo mv go /usr/local
